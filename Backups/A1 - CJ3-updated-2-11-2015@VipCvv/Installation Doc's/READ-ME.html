INSTALLING THE SCRIPT

Requirements
You will need two types of hosting for this script, "Adobe Flash Media Server Hosting" and regular "Linux based hosting".

We recommed Hostgator.com for regular hosting because our scripts were designed on Hostgator hosting.

We recommend Serverrrom.net for "Adobe Flash Media Server" hosting because of their rock solid audio/video streams and 5 star customer service.

 

Login to your servers cPanel and create a database for this script and load one of the database sql files that came with the script located in the folder named "Datbase SQL" located in the root of the script.

Upload all files onto your webserver into the public_html folder or whatever folder your plan to host the script in.
Chmod the folder named "models" to 777 (write permission).

edit "dbase.php" located in the root folder of the script and change the database information to your database information.
edit the file settings.php to your settings. Please see the Flash Media Server "READ-ME.html" file located in the "Flash Media Server Files" folder for information on setting up your RTMP address.

 

Setting up the database
import the file Database_SQL.sql located in the "Database SQL" folder using the PhpMyAdmin software on your hosting control panel.

The SQL file named "CJ3-Database-5-models-final.sql" will install 5 premade performer accounts into your database.
The SQL file named "CJ3-Database-50-models-final.sql" will install 50 premade performer accounts into your database.

by default the script displays 50 performer thumbnails per page, provided you have 50 accounts created.

 
   SETTING UP THE FLASH MEDIA SERVER

This is so easy. You will need an FTP client such as FileZilla. You can get it free here.

https://filezilla-project.org/

FTP into your Flash Media Server and open the folder named "applications". Inside that folder create a folder and name it whatever you want. (example: "adult" or "webcams").

Upload the "main.asc" file into the new folder you created. Now your Flash server is setup properly. NEXT>>>> Next you need to find your Flash hosting RTMP address. If you are using Serverroom.net for you Flash hosting then your RTMP address should look like this.

RTMP://your-account.srfms.com/

Using the text editor of your choice open the file named "settings.php" which is located in the root of the script. You will see a sample RTMP address in the file code, simply replace the sample RTMP address with your real RTMP address.

WAIT! That's NOT all....

At the very end of your RTMP address you need to add a forward slash followed by the name of the folder you created in the "applications" folder on the Flash Media Server. So lets say you used one of my examples and named your folder "webcams", your RTMP
address in the "settings.php" file should look like this.

rtmp://your-account.srfms.com/webcams/

DO NOT FORGET THE LAST FORWARD SLASH AT THE END OF THIS URL.

rtmp://your-account.srfms.com/webcams/ <<<----- This forward slash

SAVE the "settings.php" file and upload it to your regular
hosting server in the root folder of your script.

Flash Media Server is now setup.
