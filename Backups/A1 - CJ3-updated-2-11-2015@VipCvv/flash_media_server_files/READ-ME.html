SETTING UP THE FLASH MEDIA SERVER

This is so easy. You will need an FTP client such as FileZilla.
You can get it free here.
https://filezilla-project.org/

FTP into your Flash Media Server and open the folder named
"applications". Inside that folder create a folder and name it
whatever you want. (example: "adult" or "webcams"). upload the
"main.asc" file into the new folder you created. Now your Flash
server is setup properly.

NEXT>>>>

Next you need to find your Flash hosting RTMP address. If you
are using Serverroom.net for you Flash hosting then your RTMP
address should look like this.

RTMP://your-account.srfms.com/

Using the text editor of your choice open the file named "settings.php" which is located in the root
of the script. You will see a sample RTMP address in the file code,
simply replace the sample with your real RTMP address.

WAIT! That's NOT all....

At the very end of your RTMP address you need to add a forward
slash followed by the name of the folder you created in the
"applications" folder on the Flash Media Server. So lets say you
used one of my examples and named your folder "webcams", your RTMP
address in the "settings.php" file should look like this.

rtmp://your-account.srfms.com/webcams/

DO NOT FORGET THE LAST FORWARD SLASH AT THE END OF THIS URL. /

rtmp://your-account.srfms.com/webcams/ <<<----- This forward slash

SAVE the "settings.php" file and upload it to your regular
hosting server in the root folder of your script.

Flash Media Server is now setup.
